# 🏥 Cancer Patient Analytics Dashboard

An end-to-end Healthcare Analytics project built using **Python, SQL Server, and Power BI** to analyze **100,000 synthetic cancer patient records** across India from **2022–2025**.

The project demonstrates the complete analytics workflow from data cleaning and exploratory data analysis (EDA) to SQL-based business insights and interactive Power BI dashboards.

---

## 📌 Project Overview

This project was developed to uncover trends in cancer diagnosis, treatment effectiveness, patient survival outcomes, and geographic distribution of cases across India.

### Key Objectives

* Analyze cancer patient demographics
* Study survival outcomes by cancer type and stage
* Evaluate treatment effectiveness
* Identify geographic trends and hospital performance
* Create an interactive executive dashboard for decision-making
